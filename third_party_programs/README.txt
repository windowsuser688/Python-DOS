Third Party Programs Folder
===========================

This folder is for custom Python programs that you want to run from Python-DOS.

How to add your own programs:
1. Place any Python (.py) file in this folder
2. Launch Python-DOS and select "Third Party Programs" from the main menu
3. Your program will appear in the list
4. Select it and press ENTER to run it

Tips:
- Programs can be simple scripts or full curses applications
- Use the app_template.py in the root directory as a starting point
- Programs will run and then return to Python-DOS when they exit
- Make sure your program handles errors gracefully

Example:
- Copy app_template.py to this folder
- Rename it to my_program.py
- Modify it with your own logic
- Run it from the Third Party Programs menu!

There is some sample apps in it, so you can directly run it!
